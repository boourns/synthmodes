Pico Seq

Pico Seq is a 16-step clock and gate sequencer by Erica Synths, in 3hp.

When using Pico Seq, numbers 9-16 are represented as "1.-8." with the LED period lit in the bottom right.

Octave Range

Accessing

Push Step until light flashes yellow.

Operation

Turn Step to set octave range, from 1-8.


Master Pitch Slide

Description

When enabled, Master Pitch Slide enables pitch slide on random steps or on all steps.

Operation

Hold Rnd.
Turn Step to select Master Pitch Slide mode.
Display Mode
0 Disabled
1 Pitch slide random steps
"\" shape Pitch slide all steps


Step Sequencing

Enter Notes

Accessing

Push Step until light is not lit.
Push Value until light is green.

Operation

Turn Step to select sequence step.
Turn Value to select note.
LED indicator shows selected note. Period indicates note is sharp.


Gate Length/Mute/Merge

Accessing

Push Step until light is not lit.
Push Value until light is red.

Operation

Turn Step to select sequence step.
Turn Value to select gate length.

Display

As Value is set, the completed length of the circle indicates the gate duty cycle.
nothing displayed = 0% duty cycle (step mute)
Full circle = 100% duty cycle


Pitch Slide

Accessing

Push Step until light is not lit.
Push Value until light is yellow.

Operation

Turn Step to select sequence step.
Turn Value to enable pitch slide for step.

Display

"S"-like shape = slide enabled
"-|"- like shape = slide disabled


Pattern Modification

Transpose

Operation

While sequence is playing, push and hold Value.
Turn Step clockwise to transpose sequence up, counter-clockwise to transpose down.

Release Value to return to original sequence.
Press Step to overwrite sequence with transposed version.


Randomize Pattern

Operation

Press Rnd to randomize pattern within configured range and scale.


Load/Save

Load

Description

Pico Seq can store/recall 16 sequences.

Operation

Push L/S. Step will blink green.
Turn Step to select save slot.
Press Step to load sequence.


Save

Description

Pico Seq can store/recall 16 sequences.

Operation

Push L/S twice. Step will blink red.
Turn Step to select save slot.
Press Step to save sequence.
