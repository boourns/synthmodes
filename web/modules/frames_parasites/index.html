General

Info

Frames

Frames is a digitally controlled quad VCA by Mutable Instruments.

Frames Parasites is an alternate firmware by Matthias Puech, extending the original modes and adding completely new modes.

Mode Switching

To switch modes in Frames Parasites, press both Add and Del, then move one of the five knobs to select a mode. Press any button to enter tha selected mode,

Settings

Channel Settings

Interpolation curve Hold Add for 1 second.
Channel knobs control interpolation curves per channel.
Options are step, linear, accelerating, decelerating, smooth departure/arrival, and bouncing.
Response curve Hold Del for 1 second.
Channel knobs control response curves per channel, from linear to exponential.

Save/Reset State

Frames Parasites offers a 5 save slots. Knobs 1-4 select user slots,

Save Hold Add for 3 seconds, then turn a knob to select that save slot. Turning Frame knob selects the boot settings. Press Add again to confirm save, Del to cancel.
Reset Hold Del for 3 seconds, then turn a knob to select that save slot. Turning Frame knob resets the module to blank settings. Press Del again to confirm load, Add to cancel.


Modes

Frame - Keyframer

Operation

Frame knob and CV input smoothly interpolates between keyframes. To play the animation patch an LFO into Frame.


Frame CW - Autoplay Keyframer

Operation

Enter this mode by turning Frame all the way clockwise during mode switch.
Mode is same as regular keyframer mode, but animation autoplays forwards or reverse at a speed controlled by Frame CV input.


1 - Quad Sequencer

Operation

Send pulses on the Frame input to cycle output to the next keyframe. Ensure Modulation attenuverter is set to maximum. Frame knob no longer modifies output state.

Step Edition Mode is a Parasites addition where Knobs 1-4, if moved, edit the value of the step currently playing instead of the values of the keyframe selected by Frame. In this mode Frame controls number of steps in sequence. Long-press Del to enter the mode, press Del to exit.

Long press Add to randomize all values in sequence.



2 - Shift Register Sequencer

Operation

Feed steady clock into Frame CV to generate clock ticks. What happens on a given clock tick depends on Knobs 2 & 3. This mode uses programmed keyframes, but only uses the Knob 1 keyframe values.

Knob 2 controls the number of clock ticks it takes for the sequencer to step. CCW acts as clock divisor, CW randomly skips.

Knob 3 controls the number of clock ticks it takes for the outputs to shift once to thr right. CCW acts as clock divisor, CW randomly skips.

Knob 4 introduces error on the sequence values.

Fr. Step outputs a pulse when either the sequencer steps or the outputs shift.

Hold Add for 3 seconds, then use Knobs 1-4 to set pattern length. Press any button to exit.

Hold Del for 3 seconds, then use Knobs 1-4 to set envelope shape. Press any button to exit.



3 - Quad LFO

Operation

Enable +10v offset. Outputs are now wavetable oscillators.

Frame knob and CV control pitch.

Channel 1 knob controls wavetable position.

Channel 2 knob offsets wavetable position of each oscillator. At 12 o'clock all oscillators have same wave shape.

Channel 3: After 12 o'clock creates phase shift between oscillators. Before 12 o'clock changes frequency between oscillators.

Channel 4 knob adds phase-modulation between neighbouring oscillators. Before 12 o'clock to modulate from previous channel, after 12 o'clock to modulate from next channel.

Parasites Additions

Add resets oscillator phase.

Del randomizes oscillator phase.

Fr. Step is high for half the cycle of Oscillator 1.



4 - Euclidean Sequencer

Operation

Feed steady clock into Frame CV. This mode does not use programmed keyframes.

Frame knob controls rotation between channels.

Knobs 1-4 control Fill.

Hold Add for 3 seconds, then use Knobs 1-4 to set pattern length. Press any button to exit.

Hold Del for 3 seconds, then use Knobs 1-4 to set envelope shape. Press any button to exit.
